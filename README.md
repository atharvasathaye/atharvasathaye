# Atharva Sathaye

Graduate student in Information Management at the University of Illinois Urbana-Champaign. I build data pipelines, machine learning models, and full-stack analytics tools.

## Featured Projects

### [VeriStream AI](https://github.com/atharvasathaye/Cozad)
Video verification platform for deepfake detection. FastAPI backend with HuggingFace pretrained models, multi-signal frame analysis, quality gating, and a Streamlit dashboard for reviewing risk reports.

**Python, FastAPI, PyTorch, HuggingFace, OpenCV**

### [Supply Chain Pressure Analysis](https://github.com/atharvasathaye/Supply-chain)
Analyzes the relationship between global supply chain pressure (NY Fed GSCPI) and US consumer inflation (FRED CPI). Streamlit dashboard with dual-axis charts, Pearson correlation, and Power BI reports.

**Python, Streamlit, Plotly, Power BI**

### [Formula 1 Predictions](https://github.com/atharvasathaye/formula-1)
Race result prediction model built on historical F1 data from the Ergast API and FastF1 telemetry. Feature engineering combines driver performance, qualifying gaps, and track-specific factors.

**Python, scikit-learn, FastF1, Streamlit**

### [Career Insights Dashboard](https://github.com/atharvasathaye/Student-Career-dashboard)
Star schema data warehouse tracking UIUC graduate employment outcomes. SQL ETL pipeline feeding interactive Tableau dashboards. Recognized by faculty and recommended to the university career center.

**SQL, Tableau, Star Schema**

### [311 Service Requests: Differential Privacy](https://github.com/atharvasathaye/HCD_Differential_Privacy)
Epsilon-delta differential privacy applied to Chicago 311 service request data. Demonstrates privacy-utility tradeoffs across varying epsilon values while preserving aggregate spatial patterns.

**Python, pandas, NumPy, Plotly**

### [Healthcare Billing Assistant](https://github.com/atharvasathaye/ABR_Healthcare_Client)
AI-powered bilingual billing chatbot with RAG retrieval over insurance policy documents. FastAPI backend with OpenAI integration, Azure deployment, and PHI/PII safety guardrails.

**Python, FastAPI, OpenAI API, Azure**
